# Pak TierTagger

Pak TierTagger is a client-side Fabric mod that automatically fetches and displays PvP tier rankings from PakTiers directly on player nametags — so you always know who you're up against, without leaving the game.

✨ Features

  • 🏷️ Live tier tags on nametags — See every player's PvP tier displayed above their head in real time
  • 🎮 Multiple gamemode tiers — Supports Sword, Axe, Crystal, Pot, SMP, UHC, Neth Pot, Mace, and more
  • 🔍 In-game player search — Look up any player's tier without leaving the game
  • ⌨️ Keybinds — Quickly cycle gamemodes “G” or open the player search screen “Y”

📋 Requirements

  • Minecraft 1.21 – 1.21.1
  • Fabric Loader ≥ 0.16.0
  • Fabric API
  • ModMenu ≥ 11.0.3
  • Java 21+
